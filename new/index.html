2-Digit Lottery Blockchain System

Welcome to the Lottery Blockchain

This system implements a blockchain-based lottery with the following features:

  • 10 peers participating in daily lottery draws
  • Blockchain-based recording of entries, winner selection, and reward distribution
  • Commit-reveal scheme for fair randomness generation
  • Proof of Stake consensus mechanism
Open Visualization Dashboard

How to Run

To start the lottery blockchain system:

  1. Install dependencies: pip install -r requirements.txt
  2. Run the system: python main.py --visualize
  3. Or use the run script: ./run.sh
